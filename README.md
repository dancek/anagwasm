# Anagwasm

A naive browser port of [Detegr/ufag-rs](https://github.com/Detegr/ufag-rs) (a naive Rust port of [sliedes/ufag](https://github.com/sliedes/ufag)).

## Usage

Build with `./build.sh`, then run a http server in `pkg/`.
